# 🤖 CodeAlpha AI FAQ Chatbot

## 📌 Project Overview
This project is an AI-powered FAQ Chatbot developed using Python and Flask. The chatbot answers user queries by matching them with frequently asked questions using Natural Language Processing (NLP) techniques.

## ✨ Features
- Interactive chatbot interface
- Answers frequently asked questions
- Uses NLP for text similarity matching
- Fast and user-friendly system
- Web-based application using Flask

## 🛠️ Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- Scikit-learn
- JSON

## 🚀 How to Run
1. Clone or download the repository.
2. Install the required libraries:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the Flask application:
   ```bash
   python app.py
   ```
4. Open the browser and visit:
   ```
   http://127.0.0.1:5000/
   ```
5. Start chatting with the FAQ Bot.
